# Student Attendance Management System (SAMS)

## Project Overview

SAMS is a web-based system designed to digitize attendance tracking in educational institutions.

## Main Features

- Role-based login
- Student dashboard for viewing attendance
- Teacher attendance marking interface
- Admin approval system for attendance edit requests

## Files Included

- index.html
- student.html
- teacher.html
- admin.html
- style.css
- script.js
- setup.txt

## How to Run

Open index.html in any browser and test the different roles.

## Note

This project is a functional prototype demonstrating the core workflow of SAMS based on the SRS and Architecture Design Document. Backend and database interactions are simulated.
