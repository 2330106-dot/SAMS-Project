# Student Attendance Management System (SAMS)

Developed as part of Software Engineering Project (KIIT University)

## 📌 Project Overview
SAMS is a web-based application designed to digitize and simplify attendance tracking in educational institutions. The system demonstrates role-based access control and workflow management for students, teachers, and administrators.

## 🎯 Features
- Role-based login (Student, Teacher, Admin)
- Student dashboard to view attendance records
- Teacher interface to mark and manage attendance
- Admin panel for handling approval of edit requests
- Structured workflow for attendance tracking

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

- `index.html` – Login page  
- `student.html` – Student dashboard  
- `teacher.html` – Teacher interface  
- `admin.html` – Admin panel  
- `style.css` – Styling  
- `script.js` – Core logic  
- `setup.txt` – Setup instructions  

## ▶️ How to Run
1. Download or clone the repository  
2. Open `index.html` in any web browser  
3. Login as Student / Teacher / Admin to explore features
