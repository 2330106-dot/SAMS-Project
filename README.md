# Student Attendance Management System (SAMS)

**Developed as part of Software Engineering Project (KIIT University)**

## Project Overview
SAMS is a web-based system designed to digitize attendance tracking in educational institutions. It demonstrates role-based access and workflow management for students, teachers, and administrators.

## Demo Video
https://drive.google.com/file/d/1ZKfHh6mtr0xV0KqMmpG7V5sibhmsuhEN/view

## Features
- Role-based login (Student, Teacher, Admin)
- Student dashboard for viewing attendance
- Teacher attendance marking interface
- Admin approval system for edit requests

## Technology Used
- HTML
- CSS
- JavaScript

## Files Included
- index.html
- student.html
- teacher.html
- admin.html
- style.css
- script.js
- setup.txt

## How to Run
1. Open `index.html` in any browser
2. Login as Student / Teacher / Admin

## Note
This is a functional prototype based on the SRS and Architecture Design Document. Backend and database interactions are simulated for demonstration purposes.
